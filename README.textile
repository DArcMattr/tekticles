tekticles is a simple set of fonts, derived from ClearFont2.
I only ever used one set of fonts from CF2 (the "Calibri":http://en.wikipedia.org/wiki/Calibri set),
so I ripped CF2 down to it's bear ass and stripped off all the extra crap.  This addon is what's left.
